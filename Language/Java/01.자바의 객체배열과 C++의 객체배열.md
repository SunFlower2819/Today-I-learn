```Java
class Circle {
	int radius;

	Circle(int r) {
		radius = r;
	}

	double getArea() {
		return radius * radius * 3.14;
	}
}

public class test {
	public static void main(String args[]) {
		Circle c[] = new Circle[5];

		for (int i = 0; i < c.length; i++) {
			c[i] = new Circle(i + 1);
		}

		for (int i = 0; i < c.length; i++) {
			System.out.println(i + "번째 원 : " + c[i].getArea());
		}
	}
}
```

자바는 객체를 생성할 때 ``new`` 연산자를 사용한다
